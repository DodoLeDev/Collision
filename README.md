## Building

### Dependencies

- `Crystal` - `~1.2.1`
- `GTK`
- `libadwaita`

#### Manually

1. Install crystal
2. `$ shards install`
3. `$ crystal build -Dpreview_mt src/hashbrown.cr --release`

#### Makefile

1. `make`
2. `# make install` # To install it
